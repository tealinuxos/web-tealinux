# Welcome to TeaLinuxOS

**TeaLinuxOS** is a Linux distribution based on **Arch Linux**, developed by the *Dinus Open Source Community (DOSCOM)*. It is crafted with a strong focus on programming and development, providing a clean and efficient environment for developers.

## Philosophy
TeaLinuxOS carries the philosophy: *"Nikmatnya Sebuah Racikan"*, which roughly translates to **"The Delight of a Well-Crafted Blend"**.
